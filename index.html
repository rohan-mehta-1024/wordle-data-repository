Wordle Data Repository

By: Rohan Mehta

This site contains analytics data on the viral word-game Wordle. It is updated daily so that all graphics display the correct information. The code for the project was written in the Wolfram Language, and can be accessed

Please be patient with some of the graphics; they contain a lot of data, and so will not load instantly. Due to the amount of rendering required for the third visualization, it wil only display the last seven days worth of data at a time. The first two visualizations represent the entire dataset.
